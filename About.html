Autonomous Surface Vessel

About The  Webb Institute  ASV

This project is a continuation of multiple student's theses. 

In 2016 the project began with Issac Kremers designing and constructing the model and writing code. Back then, the vessel only used an arduino, so the code was written in C++. 

In 2018 this project was continued by Nicholas Ratinaud who re-wrote the code for path following.

In 2020 and 2021, Max Haland added a Raspberry Pi for the main computer to allow for more advanced algorithms and improved connectivity. The arduino is still utilized to provide real time response to control the hardware.

Currently, the project is being worked on by Kevin Becker
